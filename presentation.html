무심천 유역 강우 패턴
계층적 분류 분석

2단계 상관관계 분석을 통한 홍수 사건 특성화

📊 연구 개요

  • 연구 대상: 무심천 유역 (금천, 가덕 강우 관측소 → 흥덕교 수위)
  • 분석 기간: 2014년 ~ 2023년
  • 사건 수: 26개 임계점 초과 사건 (수위 2.34m 이상)
  • 분석 방법: 계층적 2단계 상관관계 분석

1차 분류

강우 관측소 간 패턴 상관관계

금천 ↔ 가덕

2차 분류

강우-수위 반응 특성 분석

금천/가덕 → 수위

🎯 연구 배경 및 목적

연구 배경

  • 무심천 유역: 두 개의 강우 관측소 (금천, 가덕)
  • 각 관측소가 수위에 미치는 영향이 사건마다 다름
  • 유역에 적합한 관측소 선정 및 다중 관측소 데이터 이용 시 반영 비율 최적화 필요
  • 필요성: 사건별 강우 패턴 특성 파악

연구 목적

  1. 1차 목표: 두 관측소 강우 패턴의 유사성/차이 분류
  2. 2차 목표: 각 사건에서의 관측소별 수위와의 상관관계 기반 강우 패턴 세분화
  3. 최종 목표: 두 관측소와 수위관측소 기반 무심천 유역 데이터 특성 분석

향후 연구 방향 (Future Work)

  • 수위 예측 모델에서 중요도 높은 특성 도출
  • 강우 패턴별 최적 가중치 탐색
  • 예상 강우량(금천, 가덕) 입력 시 패턴 기반 정밀 예측 구현
  • 실시간 패턴 분류 및 맞춤형 예측 시스템 구축

📊 1차 분류: 강우 관측소 간 패턴 분석

분석 대상

  • 두 관측소: 금천, 가덕
  • 비교 변수: 각 사건의 시간별 강수량 패턴
  • 분석 기간: 사건별 구간 길이 다름 (교차 전 24h + (교차~피크 시간차) + 피크 후 24h)

핵심 지표: Pearson 상관계수 (r)

r = Σ[(x_i - x̄)(y_i - ȳ)] / √[Σ(x_i - x̄)² · Σ(y_i - ȳ)²]

변수 설명:

  • x_i, y_i: 해당 시각(i)의 금천/가덕 강우량
  • x̄, ȳ: 해당 사건 분석 구간의 평균 강우량
  • 측정: 두 관측소 시계열 패턴의 선형 상관성

해석 기준:

  • r ≥ 0.7: 강한 양의 상관 → 동시강우
  • 0.5 ≤ r < 0.7: 중간 상관
  • r < 0.5: 약한 상관

왜 Pearson 상관계수인가?

  1. 패턴 유사성: 시간에 따른 강수량 변화 패턴 비교
  2. 표준화: 절대량이 아닌 변화 경향 측정 (-1 ~ +1)
  3. 시계열 상관 분석: Real-time adaptive model training technique using Pearson correlation coefficient 등 다수 시계열 데이터 분석 연구에서 활용

🔄 교차상관 (Cross-Correlation)

개념

두 시계열 데이터 간의 상관관계를 시간 지연(time lag)을 고려하여 측정

R(τ) = Σ[x(t) · y(t + τ)] / √[Σx(t)² · Σy(t)²]

왜 필요한가?

문제 상황

  • 금천과 가덕은 공간적으로 떨어져 있음
  • 같은 비 구름이라도 시간차를 두고 도달
  • 단순 동시 상관계수로는 유사 패턴 탐지 불가

해결 방법

  • 시간 지연 적용: -6h ~ +6h 탐색
  • 최대 상관계수와 지연 시간 찾기
  • "지연된 유사 패턴" 탐지 가능

1차 분류 기준

패턴 상관계수 (r) 시간 지연 (lag) 의미
동시강우 r ≥ 0.7 |lag| < 3h 두 관측소 패턴 유사, 시간차 작음
지연강우 r ≥ 0.7 |lag| ≥ 3h 패턴 유사하지만 시간차 큼
집중강우 r < 0.7 강수량 비율 > 2:1 한쪽 관측소에 집중된 강우
비정형 r < 0.7 - 복잡한 패턴, 명확한 분류 불가

🔬 2차 분류: 강우-수위 반응 특성 분석

핵심 질문

"동일한 1차 패턴(예: 동시강우)이라도,
어느 관측소의 강우가 수위에 더 큰 영향을 미치는가?"

분석 방법

각 관측소 강수량 → 수위의 교차상관 계산

1. 금천 → 수위

  • 금천 강수량 시계열
  • 수위 시계열
  • 시간 지연 0~12시간 탐색
  • 상관계수₁, 지연시간₁

2. 가덕 → 수위

  • 가덕 강수량 시계열
  • 수위 시계열
  • 시간 지연 0~12시간 탐색
  • 상관계수₂, 지연시간₂

추출 지표

  • 주도 관측소: 상관계수가 더 높은 쪽
  • 상관 강도: 강(≥0.7) / 중(0.5~0.7) / 약(<0.5)
  • 반응 시간: 빠름(<3h) / 보통(3~6h) / 느림(≥6h)
  • 균형 여부: 상관계수 차이 < threshold → 균형형

⚙️ Threshold 민감도 분석

⚠️ 중요한 점

2차 분류에서 "균형" vs "주도"를 구분하는 threshold 값은 분석자의 선택에 따라 달라질 수 있습니다.

상관계수 차이 데이터 분포
다양한 threshold 접근법 비교

검토한 방법들

접근법 Threshold 균형형 주도형 특징
전체 중앙값 (채택) 0.063 13 (50%) 13 (50%) 통계적 중립, 50/50 균형
동시강우 중앙값 0.045 11 (42%) 15 (58%) 계층적 일관성, 더 엄격
동시강우 최댓값 0.206 24 (92%) 2 (8%) 포괄적, 너무 관대

본 연구의 선택

Threshold = 0.063 (전체 데이터 중앙값)

이유: 통계적 중립성, 명확한 50/50 분할, 모든 패턴 포함

💡 주요 발견 및 의의

1. 계층적 분류의 필요성 확인

  • 1차 패턴(강우-강우)이 같아도 2차 특성(강우-수위)은 다를 수 있음
  • 예: 동시강우 19건 → 8가지 다른 수위 반응 패턴
  • 단순 강우 패턴만으로는 수위 예측 부족

2. 관측소별 영향력

  • 금천/가덕 거의 균등 (46%/54%)
  • 사건별로 주도 관측소 다름
  • 공간적 강우 분포 중요성

3. 반응 시간 다양성 및 대응 전략

  • 즉각 반응 패턴 (<3h): 즉각 대응 모델을 활용한 실시간 수위 예측
  • 느린 반응 패턴 (≥6h): Lag 시간을 반영한 지연 예측 모델 활용
  • 전략적 활용: 패턴 분류 후 적합한 예측 모델 자동 선택 시스템 구축 가능
  • 예: 즉각 반응 판단 시 → Short-term 예측 모델, 느린 반응 판단 시 → Lag-adjusted 예측 모델

4. 예측 모델 개발 기반

패턴별 특성을 반영한 맞춤형 수위 예측 모델 개발 가능

  • 금천주도 패턴 → 금천 강우 가중치 ↑
  • 균형 패턴 → 두 관측소 균등 반영
  • 반응 시간 → 예측 리드타임 설정
